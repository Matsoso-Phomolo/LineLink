# LineLink

LineLink is a remote line-house management and public room finder platform for landlords, caretakers, tenants, and room seekers around Roma and NUL.

- Local developer setup: [README_LOCAL.md](README_LOCAL.md)
- Public product overview: [README_PUBLIC.md](README_PUBLIC.md)

The repository contains a FastAPI backend, a React/Vite frontend, Alembic migrations, seed data, and deployment notes for Render and Vercel.
